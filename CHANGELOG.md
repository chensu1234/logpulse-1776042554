All notable changes to this project will be documented in this file.

## [1.0.0](https://github.com/yourusername/logpulse) (2026-04-13)

### Added

- **Real-time File Monitoring**: Poll-based file watching with rotation detection
- **Anomaly Detection Engine**: Pattern matching for security and system issues
- **Terminal UI**: Color-coded output with timestamps and log levels
- **Statistics Dashboard**: Live counters for lines, errors, warnings, and anomalies
- **Configuration System**: YAML-based config with customizable thresholds
- **Log Level Classification**: Automatic detection of error, warning, info, debug
- **Burst Detection**: Alert on consecutive error patterns
- **Security Patterns**: Built-in detection for SQL injection, XSS, path traversal, command injection
- **JSON Log Parsing**: Optional parsing of JSON-formatted logs
- **Documentation**: Comprehensive README with examples and command reference
